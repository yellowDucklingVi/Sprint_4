Проект содержит автоматизированные тесты для учебного сервиса Яндекс Самокат.

## Структура проекта
- `src/test/java/ru/yandex/praktikum/pages/` - Page Object классы
- `src/test/java/ru/yandex/praktikum/tests/` - Тестовые классы
- `pom.xml` - Конфигурация Maven

## Тесты
1. `QuestionsTest` - тестирование FAQ (8 вопросов)
2. `OrderTest` - тестирование оформления заказа с параметризацией
