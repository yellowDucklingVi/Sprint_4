# Автотесты для учебного сервиса Яндекс Самокат

Проект содержит автоматизированные тесты для учебного сервиса Яндекс Самокат.

## Технологии

| Технологии       | Версия   |
|------------------|----------|
| Java             | 11       |
| JUnit            | 4.13.2   |
| Selenium         | 3.141.59 |
| Maven            | 3.9.11   |
| WebDriverManager | 5.3.2    |


## Структура проекта
- `src/test/java/ru/yandex/praktikum/pages/` - Page Object классы
- `src/test/java/ru/yandex/praktikum/tests/` - Тестовые классы
- `pom.xml` - Конфигурация Maven

## Тесты
1. `BaseTest.java` - базовый тестовый класс с общими настройками 
2. `QuestionsTest` - тестирование FAQ (8 вопросов)
3. `OrderTest` - тестирование оформления заказа с параметризацией

## Запуск всех тестов
 mvn clean test
